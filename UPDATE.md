# Обновление опубликованного сайта до v0.3

В GitHub-репозитории, который подключён к Render:

1. Замените существующий `main.py` файлом `main.py` из этого архива.
2. `requirements.txt` можно оставить прежним или заменить версией из архива.
3. Нажмите **Commit changes**.

Render автоматически увидит новый commit и запустит Deploy.

В v0.3 добавлено:
- весь основной лист `Вводные`;
- сценарии Консервативный / Базовый / Оптимистичный;
- все параметры сроков, продаж, строительства, коммерческих расходов, налогов и финансирования;
- полный прогноз ключевой ставки (22 точки, 3 сценария);
- соцнагрузка;
- МФОЦ / офисы;
- ТЦ / коммерция ОСЗ;
- наземный паркинг;
- отдельная ставка себестоимости подземной части;
- полный редактируемый ТЭП по продуктам;
- сохранение введённых данных в браузере.

Важно: кредитные вводные уже отображаются, но точный расчёт БРИДЖ → ПФ → эскроу → проценты → LLCR ещё не подключён к итогам.
