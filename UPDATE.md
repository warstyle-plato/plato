# PLATO Development Model v0.5 — импорт калькулятора ТЭП ГлавАПУ

Добавлено во вкладку `Вводные`:
- загрузка `.xlsx` калькулятора ТЭП ГлавАПУ;
- автоматическое распознавание листа `ТЭП`;
- корректная обработка русского формата чисел:
  - `22,782 тыс.кв.м.` → `22 782 м²`;
  - `1 267,539 млн.руб.` → `1 267,539 млн ₽`;
  - `0,651 (100,0%)` → основное значение `0,651`;
- чтение листов `Машино-места` и `Параметры территории`, если они присутствуют;
- предпросмотр распознанных данных до применения;
- применение к Вводным и ТЭП только после подтверждения.

Автоматически переносятся:
- стоимость смены ВРИ;
- денежная компенсация за социальные объекты;
- фактически предусмотренные ДОУ/СОШ/поликлиника;
- квартиры: СПП → ГНС, НП → общая площадь, площадь квартир → продаваемая площадь, количество квартир;
- коммерция в МКД: СПП/НП;
- отдельная нежилая застройка (если есть);
- социальные объекты (если предусмотрены в проекте).

Важно:
- нормативная потребность в машино-местах из ГлавАПУ НЕ превращается автоматически в продаваемый подземный/наземный паркинг;
- норматив ДОО/СОШ/поликлиники хранится справочно, фактические объекты берутся из соответствующего блока ТЭП;
- при отсутствии фактических соцобъектов и наличии суммы компенсации автоматически предлагается режим `Денежная компенсация`.

Для обновления сайта замените `main.py` и сделайте Commit.
